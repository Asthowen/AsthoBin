<h3 align="center">
    <strong>AsthoBin is a simple website to share pieces of code with a URL, written in Rust.</strong>
</h3>

## Features
* Lightweight < 5MB
* Low RAM consumption
* Responsive UI
* Code coloring with [highlight.js](https://highlightjs.org/)
* Automatic deletion after 7 days
